# API Request

<br />

### Sanctions Screening API

Screen **cases** (individuals, entities, vessels, crypto wallet IDs, etc) for potential sanctions and other potential risk.

<Container>
  <Row>
    <Column>
      <Callout type="info">
        **Batch screening** <br />
        The **``` /search ```** endpoint accepts upto 1000 cases within a single
        request for batch screening. Both individual and batch searches use the
        same endpoint.
      </Callout>
    </Column>
    <Column>
      ```mdx filename="Endpoint" 
      POST https://v4.ofac-api.com/search 
      ```
    </Column>
  </Row>
</Container>
